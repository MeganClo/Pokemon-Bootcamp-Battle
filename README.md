### Description
A simple gaming application leveraging the Pok√©API and the Random User Generator API where users take turns and points are subtracted based on the move selected. The player who survives wins!

- A random opponent is assigned from the Random User API
- A random Pok√©mon is assigned to the user and the opponent
- Moves are pulled in to the user's buttons from the Pok√©API
- The user can select moves in a back and forth exchange with their opponent until one runs out of Health
- Remaining health is the surviving players high score
- The winner's high score is saved in a leaderboard
- The user can opt to play again via a modal
